## Upgrade Process

Follow these steps to update LuLu CHU safely.

<Steps>
  <Step title="Backup Your Site" icon="database">
    Create a full backup of your documentation repository.
  </Step>
  <Step title="Check Compatibility">
    Review release notes for breaking changes.
  </Step>
  <Step title="Install Update">
    <Tabs>
      <Tab title="npm" icon="package">
        ```bash
        npm install lulu-chu@latest
        npm run build
        ```
      </Tab>
      <Tab title="yarn" icon="package">
        ```bash
        yarn add lulu-chu@latest
        yarn build
        ```
      </Tab>
    </Tabs>
  </Step>
  <Step title="Test Locally" icon="play-circle">
    Run `npm run dev` and verify all pages render correctly.
  </Step>
</Steps>

<Callout kind="tip">
  Always test updates in a staging environment before production deployment. Use `lulu-chu --dry-run` to simulate changes.
</Callout>
